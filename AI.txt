Bilkul! AI ke main types ko samajhna kaafi interesting hai. AI ko generally 2 tarikay se classify kiya ja sakta hai: capabilities ke hisaab se aur functionality ke hisaab se. Main dono simple Roman English me bata raha hoon:

1Ô∏è‚É£ Capabilities ke hisaab se AI (Based on Intelligence level)

Narrow AI (Weak AI)

Ye AI sirf ek specific task ke liye design hota hai.

Example: Siri, Google Translate, Chess-playing AI

Note: Ye human level intelligence nahi rakhta.

General AI (Strong AI)

Ye AI human level intelligence rakhta hai aur har type ka kaam kar sakta hai jo human kar sakte hain.

Example: Abhi practical me exist nahi karta, sirf theory me hai.

Super AI

Ye AI human intelligence se bhi zyada powerful hota hai.

Abhi ye sirf concept hai, real life me nahi hai.



### **1Ô∏è‚É£ AI (Artificial Intelligence)**

* **Definition:** Machines ko intelligent banane ka overall concept.
* **Goal:** Machines ko human jaise **sochne, decision lene, aur problems solve karne** layak banana.
* **Example:** ChatGPT, Self-driving cars, Smart assistants
* **Relation:** Ye **umbrella term** hai jisme ML aur Deep Learning aate hain.

---

### **2Ô∏è‚É£ Machine Learning (ML)**

* **Definition:** AI ka ek subset, jisme machines **data se automatically seekhti hain** bina explicitly programming ke.
* **Goal:** Machines ko patterns aur predictions sikhaana.
* **Example:** Email spam detection, Netflix recommendation
* **Relation:** ML = AI ka ek **practical implementation** hai.

---

### **3Ô∏è‚É£ Deep Learning (DL)**

* **Definition:** ML ka ek advanced subset, jisme **neural networks** ka use karke complex patterns aur features automatically detect kiye jaate hain.
* **Goal:** Complex problems solve karna jaise image recognition, speech recognition.
* **Example:** Face recognition, Voice assistants, Self-driving car perception
* **Relation:** DL = ML ka **high-end version**, zyada powerful aur data-hungry.

---

### **4Ô∏è‚É£ Data Science**

* **Definition:** Data ko **analyze, interpret, aur insights generate** karne ka field.
* **Goal:** Data ke basis pe decision making aur predictive modeling.
* **Example:** Customer behavior analysis, Sales prediction

## **1Ô∏è‚É£ NumPy**

* **Purpose:** Numerical computing aur arrays ke liye.
* **Use:** Mathematical operations on arrays, linear algebra, matrices.
* **Basic Syntax Example:**

```python
import numpy as np

# 1D array
arr = np.array([1, 2, 3, 4])
print(arr * 2)  # Output: [2 4 6 8]

# 2D array
mat = np.array([[1, 2], [3, 4]])
print(np.dot(mat, mat))  # Matrix multiplication
```

---

## **2Ô∏è‚É£ Pandas**

* **Purpose:** Data analysis aur manipulation ke liye.
* **Use:** Tables/structured data ke sath kaam (DataFrames, CSV).

import pandas as pd

data = {'Name': ['Ali', 'Sara'], 'Age': [25, 30]}

print(df)

# Access column
print(df['Age'])

## **5Ô∏è‚É£ Scikit-learn**

* **Purpose:** Machine Learning library for ML algorithms.
* **Use:** Classification, regression, clustering.
* **Basic Syntax Example:**

```python
from sklearn.linear_model import LinearRegression
import numpy as np

X = np.array([[1], [2], [3]])
y = np.array([2, 4, 6])

model = LinearRegression()
model.fit(X, y)
print(model.predict([[4]]))  # Output: 8
```

---

## **6Ô∏è‚É£ TensorFlow**

* **Purpose:** Deep Learning / Neural Networks.
* **Use:** Build and train neural networks.
* **Basic Syntax Example:**

```python
import tensorflow as tf

# Simple linear model
X = tf.constant([[1], [2], [3]], dtype=tf.float32)
y = tf.constant([[2], [4], [6]], dtype=tf.float32)

model = tf.keras.Sequential([tf.keras.layers.Dense(1)])
model.compile(optimizer='sgd', loss='mse')
model.fit(X, y, epochs=10)
print(model.predict([[4]]))  # Output: approx 8
```

---

## **7Ô∏è‚É£ PyTorch**

* **Purpose:** Deep Learning / Neural Networks (like TensorFlow).
* **Use:** Flexible & dynamic computation graph.
* **Basic Syntax Example:**

```python
import torch
import torch.nn as nn
import torch.optim as optim

# Data
X = torch.tensor([[1],[2],[3]], dtype=torch.float32)
y = torch.tensor([[2],[4],[6]], dtype=torch.float32)

# Model
model = nn.Linear(1, 1)
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# Training loop
for epoch in range(1000):
    optimizer.zero_grad()
    outputs = model(X)
    loss = criterion(outputs, y)
    loss.backward()
    optimizer.step()

print(model(torch.tensor([[4.0]])))  # Output: approx 8
```
